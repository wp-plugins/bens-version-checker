 Checks the version numbers of wordpress sites.

== Description ==

This plugin checks the version numbers of other wordpress sites.  It will report the status of the site and its version number.  The plugin gets this information from the readme.html file of the listed sites.  The plugin uses wordpress to lookup the latest wordpress version and highlights the versions accordingly.

== Installation ==

1. Upload `version-checker` to the `/wp-content/plugins/` directory
2. Activate the plugin in the admin panel
3. Populate list of sites under the new `Version Checker` menu
